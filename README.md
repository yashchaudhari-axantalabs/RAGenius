# 🧠 RAGenius: The Next-Gen Knowledge Assistant

**RAGenius** is a state-of-the-art Retrieval-Augmented Generation (RAG) system that bridges the gap between private local privacy and high-reasoning cloud intelligence. Whether you are a privacy-focused researcher or a performance-driven developer, RAGenius provides a seamless, stunning interface to chat with your documents.

## 🌟 Key Highlights

### ⚡ Hybrid Intelligence
RAGenius doesn't just search; it understands. By combining **FAISS Semantic Search** (Vector-based) with **BM25 Keyword Search** (Term-based), we ensure that your queries find the right context every single time.

### 🎭 Choose Your Brain
*   **Privacy-First (Offline)**: Run entirely on your hardware using **Ollama**. Your data never leaves your machine.
*   **Performance-First (Online)**: Integrate **OpenAI (GPT-4)** or **Google Gemini** for world-class reasoning and speed.

### 💎 Glassmorphic Design
A premium, modern UI built with **React** and **Tailwind CSS**. Experience real-time streaming responses, smooth transitions, and a layout that feels as smart as the AI powering it.

---

## 🚀 Core Features

-   **Intelligent Re-ranking**: Multi-stage retrieval process that refines results for maximum accuracy.
-   **Multi-Format Support**: Effortlessly process PDF, TXT, and Markdown files.
-   **Streaming Response**: Real-time word-by-word generation for a snappy chat experience.
-   **Email Automation**: Integrated contact and feedback system with SMTP support.
-   **Dual-Environment Config**: Separated configuration for Users and Developers for maximum security and ease of use.
